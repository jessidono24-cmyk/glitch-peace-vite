## Part 1: The Science of Creativity

### Neuroscience of Creativity

#### Brain Networks

**Default Mode Network (DMN):**
- Active during rest, daydreaming
- Mind-wandering, imagination
- Autobiographical memory
- Self-referential thought
- Spontaneous creativity

**Executive Control Network (ECN):**
- Goal-directed thinking
- Focused attention
- Working memory
- Deliberate creativity
- Evaluation, refinement

**Salience Network:**
- Switches between DMN and ECN
- Detects important stimuli
- Emotional relevance
- Creative "aha!" moments

**Creative State:**
- Dynamic interplay between networks
- Flexible network switching
- Loosened associations (DMN)
- Focused execution (ECN)

**GLITCH·PEACE Integration:**
- Flow state = optimal network balance
- Exploration = DMN activation
- Challenge = ECN engagement
- Insight tiles = salience activation

#### Neurotransmitters

**Dopamine:**
- Novelty seeking
- Pattern recognition
- Reward for creativity
- Motivation to create

**Norepinephrine:**
- Arousal, alertness
- Focus on task
- Prevents distraction

**Serotonin:**
- Mood stability
- Emotional balance
- Sustained effort

**Acetylcholine:**
- Attention modulation
- Learning, memory
- Insight facilitation

**Optimal Creative State:**
- Moderate dopamine (curious, not manic)
- Optimal arousal (engaged, not anxious)
- Balanced serotonin (positive, not flat)
- Sufficient acetylcholine (focused, receptive)

#### Divergent vs. Convergent Thinking

**Divergent (Exploration):**
- Generate multiple ideas
- Think broadly
- Make unusual connections
- Suspend judgment

**Convergent (Selection):**
- Narrow down options
- Evaluate solutions

- Choose best approach

**Creative Cycle:**
1. Diverge (explore possibilities)

**GLITCH·PEACE:**
