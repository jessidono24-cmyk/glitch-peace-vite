## Design Principles

1. **Sterilized Wisdom** - No dogma, simulation framing only
2. **Non-Prescriptive** - Players discover their own path
3. **Gentle Guidance** - Suggestions, not commands
4. **Multiple Perspectives** - Many valid viewpoints
5. **Experience-Based** - Direct knowing vs. belief
6. **Integration Focus** - Understanding matters more than peak experiences
