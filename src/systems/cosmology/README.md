## Design Principles (Critical)

1. **Sterilized Wisdom** - NEVER claim metaphysical truth
2. **Simulation Framing** - "In this game/simulation/model..."
3. **Educational Approach** - "This tradition believes..."
4. **Respectful Representation** - Honor source traditions
5. **No Dogma** - Present as tools, not truth
6. **Player Interpretation** - Meaning is personal
7. **Multiple Traditions** - Show diversity, no hierarchy
